# 学习进度清单

## 第 1-2 周

- [ ] 我能运行 Python 脚本。
- [ ] 我理解变量、列表、字典、循环和函数。
- [ ] 我能用 NumPy 创建数组并计算均值、标准差。
- [ ] 我能用 pandas 读取 CSV。
- [ ] 我能画出一条或多条光谱曲线。

## 第 3-5 周

- [ ] 我理解特征 `X` 和标签 `y`。
- [ ] 我能划分训练集和测试集。
- [ ] 我能训练 Logistic Regression、SVM 或 Random Forest。
- [ ] 我能解释 accuracy、precision、recall、F1。
- [ ] 我能画出混淆矩阵。
- [ ] 我能用 PCA 做降维和可视化。

## 第 6-8 周

- [ ] 我理解 Tensor、Dataset、DataLoader。
- [ ] 我能写出一个 PyTorch 训练循环。
- [ ] 我能训练一个简单 CNN。
- [ ] 我能读取图像并做 resize、归一化。
- [ ] 我能分析模型的错分样本。
- [ ] 我理解冷冻电镜图像低信噪比、低对比度和漂移问题。
- [ ] 我能对 cryo-EM 灰度图做归一化、去噪和对比度增强。
- [ ] 我能计算局部强度、标准差和边缘变化来描述细胞相位变化。

## 第 9-10 周

- [ ] 我理解光谱平滑、标准化、SNV 和基线校正的目的。
- [ ] 我能比较不同预处理对分类效果的影响。
- [ ] 我能使用 PCA + SVM 做光谱分类。
- [ ] 我能找出模型认为重要的波段。

## 第 11-12 周

- [ ] 我完成了一个端到端小项目。
- [ ] 我记录了数据来源和预处理方法。
- [ ] 我报告了模型指标和混淆矩阵。
- [ ] 我写出了错误分析。
- [ ] 我能说明 cryo-EM 相位变化结果中哪些是模型证据，哪些只是推测。
- [ ] 我写出了项目局限性和下一步计划。
